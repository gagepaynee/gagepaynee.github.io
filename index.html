Alexa Skill

Bible Topics

Bible Topics is an Alexa Skill that allows the user to request a topic and then returns a relevant Bible Verse to them. It keeps track of user upvotes and downvotes to make sure that it returns only the most enjoyed/relevant bible verses for specific topics and will never return a disliked/irrelevant verse to the user again for that topic. It also allows for a user to request a specific verse or passage from the Bible.

  • Node.js
  • DynamoDB
  • Alexa Skill Kit
  • AWS Lambda
